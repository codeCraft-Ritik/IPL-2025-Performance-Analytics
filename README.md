# 🏏 IPL 2025 Performance Analytics Dashboard

An interactive **Data Analytics and Visualization project** that analyzes IPL 2025 player performance using batting and bowling datasets.
This project extracts meaningful insights from cricket statistics and presents them through **interactive charts and a dynamic Streamlit dashboard**.

---

# 🚀 Project Overview

The **IPL 2025 Performance Analytics Dashboard** provides a detailed analysis of:

* 🔥 Top Run Scorers
* ⚡ Best Strike Rates
* 🎯 Top Wicket Takers
* 💰 Best Economy Bowlers
* 🌟 Best All-Rounders
* 📊 Interactive data visualization

The dashboard helps explore **player performance insights** through visually appealing charts and tables.

---

# 📊 Dashboard Features

### 🔥 Batting Analysis

* Top Run Scorers
* Best Strike Rate Players
* Highest Batting Averages

### 🎯 Bowling Analysis

* Top Wicket Takers
* Best Economy Rates
* Best Bowling Averages

### 🌟 All-Rounder Performance

Players appearing in both batting and bowling datasets are evaluated using a **custom All-Rounder Score**.

### 📈 Interactive Visualizations

Charts built using **Plotly** for interactive exploration.

# ⚙️ Installation

Clone the repository

```
git clone https://github.com/codeCraft-Ritik/IPL-2025-Performance-Analytics.git
```


Install required libraries

```
pip install streamlit pandas plotly
```

---

# ▶️ Run the Dashboard

Run the Streamlit application

```
streamlit run dashboard.py
```

The dashboard will open in your browser.
